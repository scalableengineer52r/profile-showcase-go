# Profile showcase (Go)

A minimal, privacy-first profile generator written in Go. It renders a concise, professional README-style Markdown from a YAML configuration without any personal identifiers.

## Features
- Modular sections: headline, skills, projects, highlights, badges
- Privacy-first: zero personal data; purely technical content
- Single binary CLI for fast rendering
- Clean, deterministic output for version control
